Internal data used by the editor.

For auto-layer tiles: "[ruleId, coordId]".

For tile-layer tiles: "[coordId]".

f

f: number
  • Defined in typedef.ts:409

"Flip bits", a 2-bits integer to represent the mirror transformations of the tile.

  • Bit 0 = X flip
  • Bit 1 = Y flip

Examples:

  • f == 0 -> no flip
  • f == 1 -> X flip only
  • f == 2 -> Y flip only
  • f == 3 -> both flips
